# Event Manager Panel

Una aplicación React para gestionar eventos usando una API mock.

## Características

- Listar eventos
- Crear, editar y eliminar eventos
- Vista detallada
- Búsqueda por nombre
- Diseño responsive con TailwindCSS

## Tecnologías

- React.js + Hooks
- React Router
- Axios
- TailwindCSS
